# curl

URL：统一资源定位符，Uniform Resource Locator

Command Line URL viewer

> curl 是常用的命令行工具，用来请求 Web 服务器。它的名字就是客户端（client）的 URL 工具的意思。

## 常见用法

直接在curl命令后加上网址，就可以看到网页源码。
